# ADHD-планировщик

Простое PWA-приложение для планирования дня. Работает полностью офлайн, ставится на домашний экран, все данные хранятся только в браузере (IndexedDB). Без сервера, регистрации и облака.

**Живое приложение:** [https://tmksdm.github.io/adhd/](https://tmksdm.github.io/adhd/)

## Возможности

- Добавление задачи в два клика (нужно только название).
- Два режима: «Сейчас» (одна текущая задача крупно + быстрый ввод) и «Сегодня» (полный список).
- Перемещение задач вверх/вниз, редактирование названия, отметка выполненных.
- Счётчик серии (streak) с приятной анимацией. Нет наказания за пропуски.
- Светлая и тёмная темы.
- Экспорт и импорт всех данных одним JSON-файлом (единственный способ перенести данные между устройствами).

## Стек

Vite + React + TypeScript, Tailwind CSS, Dexie.js (IndexedDB), vite-plugin-pwa.

## Разработка

```bash
npm install
npm run dev      # localhost:5173/adhd/
npm run build    # сборка в dist/
```

Деплой автоматический: пуш в ветку `master` → GitHub Actions → GitHub Pages.
