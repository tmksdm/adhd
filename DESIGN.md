# DESIGN.md — Паспорт стиля «ADHD-планировщик»

> Этому файлу UI следует ВСЕГДА. Любое окно/компонент сверяется с ним.
> Цель: единый стиль во всём приложении, без generic-AI-вида.

## 0. Концепция
**Calm Dark + Bento-зонирование.** Спокойная база с низкой стимуляцией.
Экран собран из крупных понятных зон-«плиток». Яркий «дофаминовый» акцент
вспыхивает ТОЛЬКО в момент награды (выполнение задачи, streak).
Обязательны ДВЕ темы: тёмная (по умолчанию) и светлая — выбор пользователя.

## 1. Принципы (из продуктовой ДНК)
- Низкий визуальный шум: мало цветов, много «воздуха», спокойные переходы.
- Время — визуально (таймлайн), а не цифрами в списке.
- Крупные зоны касания под большой палец.
- Без чувства вины: невыполненное НЕ краснеет.
- Награда = единственный момент яркости и движения.

## 2. Темы и токены (CSS-переменные)
Темы переключаются классом на <html>: `.dark` / без него (светлая).
Все цвета задаём как HSL-токены, компоненты используют только токены.

### Тёмная тема (по умолчанию)
- `--bg`            222 18% 9%    /* почти-чёрный уголь, НЕ #000 */
- `--surface`       222 16% 13%   /* плитки/карточки */
- `--surface-2`     222 14% 17%   /* вложенные блоки, поля ввода */
- `--border`        222 12% 22%
- `--text`          210 15% 92%   /* основной текст */
- `--text-muted`    214 10% 62%   /* второстепенный */
- `--reward`        74 95% 55%    /* ЛАЙМ-неон — только награда */
- `--reward-soft`   74 60% 28%    /* тусклый лайм для «спящего» состояния */

### Светлая тема
- `--bg`            210 30% 97%
- `--surface`       0 0% 100%
- `--surface-2`     210 25% 95%
- `--border`        214 20% 88%
- `--text`          222 22% 16%
- `--text-muted`    220 10% 45%
- `--reward`        82 75% 38%    /* приглушённый оливково-зелёный, не слепит */
- `--reward-soft`   82 45% 78%

> Правило: акцент `--reward` НЕ используется для обычных кнопок, ссылок, фона.
> Только: кнопка «Выполнено», анимация награды, счётчик streak.

## 3. Типографика
- Шрифт интерфейса: **Inter** (system-ui как запасной).
- Заголовок экрана: 28px / 700 / плотный межбукв. -0.02em.
- Крупная задача (режим «Сейчас»): 32–40px / 700.
- Текст задачи в списке: 17px / 500.
- Второстепенный/время: 14px / 500 / цвет `--text-muted`.
- Межстрочный: 1.4.

## 4. Шкала отступов (spacing)
Кратно 4px: 4, 8, 12, 16, 20, 24, 32, 48.
- Внешние поля экрана: 20px по бокам.
- Внутренние поля плитки: 20px.
- Промежуток между плитками: 12px.

## 5. Радиусы (скругления)
- Плитки/карточки (bento): 24px.
- Кнопки: 16px.
- Поля ввода: 14px.
- Маленькие чипы/бейджи: 999px (полностью круглые).

## 6. Кнопки (крупные, под большой палец)
- Главная кнопка (добавить задачу, выполнено): высота 56px, текст 17px/600.
- Вторичная кнопка: высота 48px.
- Минимальная зона касания любого интерактива: 44×44px.
- Главная кнопка действия — заметная, но НЕ цвета награды (награда священна).

## 7. Анимация
- Обычные переходы: спокойные, 150–250ms, плавная кривая (ease-out).
- Появление плиток/списка: лёгкое, без отвлекающих эффектов.
- МИКРОАНИМАЦИЯ НАГРАДЫ (исключение, ей можно быть яркой):
  при выполнении задачи — вспышка/пульс цвета `--reward`, лёгкие частицы/«конфетти»
  или масштаб 1→1.08→1, 400–600ms. Streak увеличивается с короткой пульсацией.
- НИКОГДА: мигание, тряска, резкие движения у НЕвыполненных задач.

## 8. Запреты (чтобы не было generic-AI-вида)
- Не использовать дефолтные цвета shadcn «как есть» — только наши токены.
- Не делать всё фиолетово-синим градиентом.
- Не подсвечивать просрочку красным.
- Не лепить тени везде; тени мягкие и редкие.
